Color Guessing Game

In the header, you'll see an RGB color code.
In case you don't know what RGB is, it stands for Red-Green-Blue;
and then comes the 3 numbers, which are the portion of each element color;
each number takes the minimum value of 0, and maximum of 255.
What you have to do, is somehow use the given RGB code to calculate the result color,
and then select from those choices of color below.

Don't worry if you got it wrong several times, there is no penalty in this game.
Just try to have fun with the game.

You can start a new game, to get a new color to guess by clicking New Color button anytime.
To set difficulty, simply click on the Difficulty button.
